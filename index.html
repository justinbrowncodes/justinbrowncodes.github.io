Projects

Heart Failure Predictor

TASK: Create a model for predicting mortality caused by heart failure.
A dataset contains 12 features than can help predict a patient's death by heart failure. My goals were to clean the data, engineer the most relevent features, and implement the following methods:

Models:

  • Logistic Regression
  • K-Nearest Neighbor
  • Decision Tree
  • Naive Bayes
  • Random Forest
  • Support Vector Machine

Performance Metrics:

  • Accuracy
  • Precision
  • Recall
  • F1-Score


My results not only illustrate the highest performing classifiers, but also provide general insight behind correlations within a patient's data.
